# ⚙️ FisioApp - Backend API

API RESTful desenvolvida em Node.js para gerenciamento de procedimentos de fisioterapia.  
Projeto criado como parte do desafio da disciplina de Coding: Mobile.

---

## 🚀 Tecnologias Utilizadas

- Node.js
- Express.js
- MongoDB
- Mongoose (ORM)
